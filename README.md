Example of Token-based authentication in [AngularJS](http://angularjs.org) with [Express](http://expressjs.com).

### Build and Run

First, install dependencies using npm:

```sh
npm install
```

Run the example:

```sh
node auth.server.js
```

and go to [localhost:8080](http://localhost:8080).
